# ربات آپلودر تلگرام (Force-Join)

## چیکار می‌کنه؟
- ادمین فایل رو برای ربات می‌فرسته
- ربات فایل رو توی یه کانال لاگ ذخیره می‌کنه و یه لینک اختصاصی می‌سازه
- کاربر روی لینک کلیک می‌کنه؛ اگه عضو کانال‌های اجباری نباشه، دکمه‌ی جوین نشون داده می‌شه
- بعد از عضویت، فایل براش ارسال می‌شه

## دستورات ادمین (فقط برای ADMIN_ID کار می‌کنه)
- ارسال هر فایلی (عکس، ویدیو، سند، صدا) در چت خصوصی با ربات → لینک اختصاصی می‌سازه
- `/addchannel <chat_id> <join_link>` → اضافه کردن کانال جوین اجباری
  مثال: `/addchannel -1002121465833 https://t.me/+9rn5ZUdD-YdiNjdk`
- `/delchannel <chat_id>` → حذف کانال جوین اجباری
- `/channels` → نمایش لیست کانال‌های جوین اجباری
- `/stats` → نمایش آمار (تعداد کاربران، فایل‌ها، کانال‌ها)

## مراحل دیپلوی روی Render

1. یک ریپازیتوری گیت‌هاب بساز و همین سه فایل (`bot.py`, `requirements.txt`, `README.md`) رو توش آپلود کن
2. توی Render:
   - `New +` → `Web Service`
   - ریپازیتوری گیت‌هابت رو انتخاب کن
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python bot.py`
   - **Instance Type:** Free
3. توی بخش **Environment Variables**، این‌ها رو اضافه کن:

   | Key | Value |
   |---|---|
   | `BOT_TOKEN` | توکن ربات از BotFather |
   | `DATABASE_URL` | همون External Database URL از Render Postgres |
   | `LOG_CHANNEL_ID` | آیدی کانال لاگ (با `-100` شروع بشه) |
   | `ADMIN_ID` | آیدی عددی خودت |

4. روی **Create Web Service** بزن و صبر کن دیپلوی تموم بشه
5. بعد از دیپلوی، این دستورات رو توی چت با ربات بزن تا کانال‌های جوین اجباری اضافه بشن:
   ```
   /addchannel -1002121465833 https://t.me/+9rn5ZUdD-YdiNjdk
   /addchannel -1001979449791 https://t.me/ava_teo
   ```
   (لینک‌ها رو با لینک واقعی خودت جایگزین کن اگه فرق داره)

## نگه داشتن ربات بیدار (Free Plan)
Render بعد از ۱۵ دقیقه بی‌فعالیتی سرویس رایگان رو می‌خوابونه. برای جلوگیری:
1. یه اکانت رایگان توی **uptimerobot.com** بساز
2. یه Monitor از نوع HTTP(s) بساز که هر ۵ دقیقه به آدرس سرویس Render‌ات (همون که با `.onrender.com` تموم می‌شه) پینگ بزنه

## نکات امنیتی
- بعد از استقرار موفق، توکن ربات و پسورد دیتابیس رو (اگه جایی به اشتراک گذاشته شدن) از نو تولید کن
- ربات باید در همه‌ی کانال‌های جوین اجباری و کانال لاگ **ادمین** باشه
